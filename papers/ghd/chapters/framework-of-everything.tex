
\chapter{The Framework of Everything}


\section{Primitive Ontology}

We assume the existence of a configuration space $\mathcal{C}$ consisting of finite bitstrings:
\[
\mathcal{C} = \{0,1\}^n
\]

No spacetime, dynamics, or physical law is assumed.

A configuration is a complete static object.

It should be noted that the bitstring plays no fundamental role; binary system is chosen simply for the sake of convenience.



\section{Descriptions}

Each configuration $c \in \mathcal{C}$ admits a set of representations:
\[
\mathcal{D}(c)
\]

A description $d \in \mathcal{D}(c)$ is any encoding that reproduces $c$ under interpretation.

We assume representation non-uniqueness:
different $d$ may encode the same $c$.


\section{Representation Length}

We define a structural ordering over representations.

Let $[d]$ denote the equivalence class of all representations describing the same configuration.

We define a partial order $[d_1] \prec [d_2]$ if $[d_1]$ admits a strictly more coherent internal constraint structure than $[d_2]$, meaning its internal correlations can be generated from a smaller set of consistent relational rules.

Representation length $K_{\mathrm{rep}}$ is then the induced measure rank under this ordering.



\section{Structured Representation Complexity}

We refine the representation length of a description $d$ by introducing an explicit decomposition into structural components.

Each description $d \in \mathcal{D}(c)$ is assumed to admit a factorisation:

\[
d \equiv (d_{\text{spec}}, d_{\text{rel}}, d_{\text{sym}})
\]

corresponding to:
\begin{itemize}
\item spectral content,
\item internal constraint structure,
\item symmetry and indistinguishability structure.
\end{itemize}

The total representation complexity is defined as:

\[
K(d) = K_{\text{spec}}(d_{\text{spec}}) + K_{\text{rel}}(d_{\text{rel}}) + K_{\text{sym}}(d_{\text{sym}})
\]


\subsection*{Spectral Complexity}

The spectral component of the complexity is given by:

\[
K_{\text{spec}} =
\sum_{i}
\left[
\log_2\!\left(\frac{A_i}{\Delta A_i}\right)
+
\log_2\!\left(\frac{2\pi}{\Delta \phi_i}\right)
+
\log_2\!\left(\frac{\omega_i}{\Delta \omega_i}\right)
\right]
\]

where:
\begin{itemize}
\item $i$ indexes spectral modes,
\item $A_i$ is the amplitude of mode $i$,
\item $\phi_i \in [0, 2\pi)$ is the phase of mode $i$,
\item $\omega_i$ is the frequency of mode $i$,
\item $\Delta A_i$ is the required precision for distinguishing amplitudes,
\item $\Delta \phi_i$ is the required phase resolution,
\item $\Delta \omega_i$ is the frequency resolution,
\item $\log_2(\cdot)$ measures the number of bits required to specify the parameter at the given precision.
\end{itemize}

The cost of specifying a parameter is determined not by its absolute range, but by the precision required to distinguish it within the representation.

The resolution parameters $(\Delta A_i, \Delta \phi_i, \Delta \omega_i)$ are not fixed a priori, but are determined by the minimal precision required to distinguish configurations within the representation.

This term captures the distributed wave-like structure of the description.

\subsection*{Relational Complexity}

\[
K_{\text{rel}} = \text{Cost of internal constraint structure among spectral components}
\]

where:
\begin{itemize}
\item this term measures how strongly spectral components are constrained to form coherent structures,
\item it reflects compatibility, persistence, and stability of composite patterns,
\item it does not introduce additional primitives, but captures structure implicit in the description itself.
\end{itemize}

This component is responsible for:
\begin{itemize}
\item effective locality,
\item structural stability,
\item inertia-like resistance to reconfiguration.
\end{itemize}

\subsection*{Symmetry and Indistinguishability Complexity}

\[
K_{\text{sym}} = \text{Cost of encoding indistinguishable or equivalent excitations}
\]

where:
\begin{itemize}
\item this term measures the cost of representing multiple occurrences of structurally equivalent spectral components,
\item it captures invariance under permutation of equivalent components,
\item it encodes how occupancy of identical or nearly identical spectral states is represented.
\end{itemize}

Different scaling behaviors of this term give rise to distinct statistical regimes:
\begin{itemize}
\item configurations allowing unrestricted occupancy correspond to bosonic behavior,
\item configurations in which multiple occupancy is strongly suppressed correspond to fermionic behavior.
\end{itemize}

This term therefore determines the statistical structure and taxonomy of emergent excitations.


\section{Induced Measure}

Configurations acquire weight through the multiplicity of their descriptions:

\[
W(c) = \sum_{d \in \mathcal{D}(c)} e^{-K(d)}
\]

This defines an induced measure over configuration space:

\[
P(c) = \frac{W(c)}{\sum_{c' \in \mathcal{C}} W(c')}
\]

No external probability distribution is assumed.


\section{Observer Functional}

An observer is not a configuration but a functional:
\[
\mu_O : \mathcal{C} \rightarrow [0,1]
\]

$\mu_O(c)$ measures the degree to which configuration $c$ instantiates a coherent observer structure.

This includes:
\begin{itemize}
\item structural stability
\item internal consistency
\item persistence of encoded memory
\end{itemize}

The observer is therefore a graded property of configurations, not a binary predicate.

\section{Observer Conditioning}

The experienced distribution over configurations is:

\[
P(c \mid O) =
\frac{\mu_O(c)\, W(c)}{\sum_{c' \in \mathcal{C}} \mu_O(c')\, W(c')}
\]

This defines the effective reality accessible to an observer.

\section{Dual Representation Principle}

Each configuration admits two complementary descriptions:

\begin{itemize}
\item Spectral representations: capturing global structure via distributed modes
\item Geometric representations: capturing localized coherence and boundaries
\end{itemize}

Neither representation is fundamental; both are interpretations of the same underlying configuration.

Observer structure requires compatibility between both representations.

\section{Observer Compatibility}

A configuration contributes significantly to observer experience only if:

\begin{itemize}
\item it admits compressible spectral representations
\item it supports stable geometric structure corresponding to an observer functional
\end{itemize}

Formally:
high contribution to $P(c \mid O)$ requires both low $K(d)$ and high $\mu_O(c)$.

\section{Emergence Principle}

The observer-conditioned measure is dominated by configurations that maximize:

\begin{itemize}
\item representational compressibility
\item structural coherence
\end{itemize}

As a result, observer-accessible reality appears:
\begin{itemize}
\item smooth
\item stable
\item law-like
\end{itemize}

These properties are not fundamental, but induced by the structure of the measure.

\section{Internal Ordering}

Time is not a fundamental dimension.

For a given observer $O$, define an internal ordering relation:

\[
c_1 \prec_O c_2
\]

if configuration $c_2$ contains a more developed or stable instantiation of $\mu_O$ than $c_1$.

Time is therefore an emergent ordering induced by observer structure, not a physical evolution parameter.

\section{Principle of Internal Describability}

Reality is not governed by external laws or dynamics.

Instead, the experienced structure of reality is determined by:

\begin{quote}
the distribution of configurations weighted by the joint effect of representational compressibility and observer coherence.
\end{quote}

Observers exist as stable high-weight structures within this distribution.


\section{The Path Measure and Physical Laws}

By combining the induced measure of configurations with the observer functional, we arrive at the fundamental equation of the theory. If we define a history $\gamma$ as a sequence of observer-compatible configurations, the probability of inhabiting a specific history is:

\begin{equation}
\mathbb{P}(\gamma \mid O) = \frac{1}{Z_O} \exp\left(-\lambda\mathcal{C}_O[\gamma]\right), \qquad \gamma \in \Gamma_O
\end{equation}

where:
\begin{itemize}
    \item $Z_O$ is the partition function (the sum over all observer-compatible histories).
    \item $\lambda$ is the global selection pressure (compression factor).
    \item $\mathcal{C}_O[\gamma]$ is the joint spectral and geometric complexity.
\end{itemize}

Under this measure, \textbf{observed physical laws are simply the large-deviation minimizers of $\mathcal{C}_O[\gamma]$}. The universe does not follow laws because they are imposed from the outside; it appears law-like because configurations that deviate from minimal complexity are exponentially suppressed in the measure.
