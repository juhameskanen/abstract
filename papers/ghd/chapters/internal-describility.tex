\chapter{Self-Interpretive Information Principle}

\section{The Interpretation Problem}

Let us stress-test the claim of the ``unreasonable effectiveness of mathematics in the natural sciences.''

Two intelligent agents, located at different points in space and time, independently develop the same arithmetic.
Without exchanging information, both agree on basic structure:

Let Alice use a decimal system, motivated by biological convenience (ten fingers, ten toes):

\[
2 + 3 = 5.
\]

Let Bob, working in a computational environment, adopt binary representation:

\begin{itemize}
\item $2 \rightarrow 10_2$
\item $3 \rightarrow 11_2$
\item $5 \rightarrow 101_2$
\end{itemize}

In binary arithmetic, the same relation becomes:

\[
10_2 + 11_2 = 101_2.
\]

If Alice were to encounter Bob's notation without context, she would likely misinterpret the symbols.
In particular, $10_2$ does not refer to ten in decimal, but to two in base-10 interpretation.

Even if the full expression is given, interpretation still depends on the assumed encoding scheme.

Thus, meaning is not intrinsic to symbol structure alone, but depends on an interpretive convention.

\subsection*{Symbolic Relativity}

The arbitrariness of representation becomes clearer when we replace digits entirely.
For example, the binary alphabet $\{0,1\}$ may be replaced with the faces of a coin $\{H,T\}$:

\[
HT + HH = HTH.
\]

Similarly, operational symbols such as $+$ and $-$ are not inherently tied to addition or subtraction. They are part of a chosen syntactic convention.

\subsection*{Two's Complement}

In finite digital systems, identical bit patterns admit multiple semantic interpretations.

For an 8-bit word:

\begin{itemize}
\item $11111111$ may represent $255$ in unsigned interpretation
\item or $-1$ in two's complement representation
\end{itemize}

The physical state is identical; only interpretation differs.


\subsection*{Reading Order and Indexing}

Even ordering conventions are not intrinsic to the data.

A bitstring such as $001$ may correspond to different values depending on the bit ordering conventions. For example:

\begin{itemize}
\item interpreted left-to-right: $1$
\item interpreted right-to-left: $4$
\end{itemize}

and these are just two of the $n!$ possible permutations of the bit significance.


\section{Code as Data}

Consider a computer in full physical detail. A memory dump yields a finite sequence of bits. Likewise, the CPU, instruction set architecture, and storage devices are themselves physically instantiated information structures.

There is no intrinsic separation between ``code'' and ``data'' at the level of raw physical state. This distinction arises only through an interpretive layer imposed by an executing system.

Even when the architecture is known, the global state of a machine remains a configuration within a finite state space of size:

\[
2^n,
\]

where $n$ is the number of bits describing the system state.

Each configuration may participate in multiple possible interpretations depending on how state transitions are mapped to computational semantics.

From this perspective, computation is not a property of isolated states, but of relations between states under an interpretation rule.

\section{Interpretational Explosion}

As the number of bits increases, the space of possible interpretations grows combinatorially.
A single finite configuration admits a vast number of syntactically valid decompositions into:

\begin{itemize}
\item code vs data partitions
\item memory layouts
\item instruction interpretations
\item execution histories
\end{itemize}

While the state space itself contains only $2^n$ configurations, the number of possible transition functions (i.e., possible “laws” or “programs” that could govern the system) is $(2^n)^(2^n)$. Even for small systems this number is immense. For n=3 bits it is already $8^8$ = 16,777,216.
For n=10 bits the number has thousands of digits.
This vast explosion of possible interpretive rules is why neither a “Universe” nor a “Computer” can be regarded as fundamental — the space of possible rules dwarfs the system they act upon.

\section{Conclusion}

When humans interpret the output of a machine, it relies on shared conventions regarding encoding, display, and measurement.
Not even a monitor would display a circle unless mutually agreed interpretations.
It assumes a two-dimensional array of values which is interpreted as a circle under agreed geometric conventions.

If those conventions are altered, the same underlying data may appear entirely different.

Thus, the effectiveness of mathematics in describing physical systems does not imply that mathematical symbols carry intrinsic meaning.
Rather, it reflects the existence of structural correspondences between different levels of interpretation.

\begin{tcolorbox}[title=Self-Interpretive Information Principle]
Information contains within itself the full space of possible interpretations.
\end{tcolorbox}
